# Hack the Garden

This repository is used to facilitate hackathon events for the Gardener community.
It does not contain any code that is relevant for the Gardener project itself.
Instead, [issues](https://github.com/gardener/hackathon/issues) are used to submit topic proposals for hackathon events, which are then put up to vote using [discussions](https://github.com/gardener/hackathon/discussions).

> [!NOTE]
> Please only submit issues as topic proposals when you're actively involved in the hackathon event. You'll find more information below.

Are you interested in joining a Gardener hackathon event?
Head over to [our website](https://gardener.cloud/community/hackathons/) to find out about past and planned events.
The page also contains information on what to expect from a hackathon and how to get involved.

Make sure to join the [#hack-the-garden](https://gardener-cloud.slack.com/archives/C0531FVMZFU) channel in [Gardener's Slack workspace](https://join.slack.com/t/gardener-cloud/shared_invite/zt-33c9daems-3oOorhnqOSnldZPWqGmIBw).

## Submitting a topic proposal

If you're participating in a hackathon event and want to submit a topic proposal, please create an issue using [the template](https://github.com/gardener/hackathon/issues/new?template=hackathon-topic.md).
